# 🌐 Global Trading Agent Hub

A professional-grade Forex trading dashboard and multi-agent management system built with Streamlit.

## 🚀 Features
- **Multi-Agent Comparison:** Track and compare performance across different trading algorithms (e.g., Tice Sniper Mode).
- **MT5-Style Visualization:** Real-time candlestick, line, and area charts with technical indicators (EMA 20/50, RSI).
- **Auto-Update:** Live server refresh every 30 seconds for real-time market monitoring.
- **Risk Management:** Built-in 2% equity cap calculator for disciplined position sizing.
- **Unified Ledger:** Comprehensive history of all executed trades and floating profits.

## 🛠️ Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Honcho92/global-trading-hub.git
   cd global-trading-hub
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the dashboard:
   ```bash
   streamlit run dashboard.py
   ```
